SSML in HTML Example Cases

The following are three alternatives for bringing SSML into HTML.

  1. The data attribute model relies on a single attribute model to represent the desired SSML feature. The attribute name is fairly meaningful
  2. The SSML as name-spaced SSML attributes is less intuitive. While the attribute names are unique across the SSML elements, they lack the contextual meaning provided by the SSML element to which they were originally defined.
  3. The CSS3 Speech approach does not support phonetic spelling or substitution.
